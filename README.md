# coffeeblog

A coffee blog with full CRUD with eight different types of coffee to provide image, quantity, name, description, and price. Customers can access to edit, delete, and buy.

---

### Technologies Used
 - HTML
 - CSS
 - Node.js
 - Mongoose
 - MongoDB
 - EJS
